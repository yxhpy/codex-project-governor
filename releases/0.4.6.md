# 0.4.6 — Clean Reinstall Manager

## Added

- `clean-reinstall-manager` skill.
- User-level full reinstall command generator.
- Governed-project discovery helper for non-project execution.
- Clean project refresh helper that re-runs project-owned initialization safely.
- Trash/quarantine workflow for plugin-global noise accidentally copied into projects.

## Behavior

- Outside a governed project, the orchestrator stops and lists discovered Project Governor projects.
- When the requested project is the Project Governor plugin root, refresh stops without quarantining plugin-owned assets.
- Inside a governed project, it generates a refresh plan and can apply safe additions/section merges/quarantine operations.
- Memory and decision files are never overwritten.

## Compatibility

- No new third-party dependencies.
- Additive release; project migration is optional and user-triggered.
