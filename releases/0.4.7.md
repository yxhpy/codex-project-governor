# 0.4.7 — DESIGN.md Governor

## Added

- `design-md-governor` skill.
- Bundled deterministic DESIGN.md linter, summarizer, and diff helper.
- Opt-in DESIGN.md template under `managed-assets/design-md/`.
- DESIGN.md prompt template and examples.
- Targeted tests for lint, summarize, and diff behavior.

## Behavior

- Does not create or overwrite project `DESIGN.md` automatically.
- Keeps Google-style design support in plugin-managed assets unless the user opts in.
- Provides fallback Python validation when the official `@google/design.md` CLI is unavailable.

## Compatibility

- No new third-party Python dependencies.
- Additive release; project migration is optional and user-triggered.
