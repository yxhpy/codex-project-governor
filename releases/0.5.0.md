# 0.5.0 — GPT-5.5 Auto Orchestration

## Added

- `gpt55-auto-orchestrator` skill.
- `context-indexer` skill.
- GPT-5.5-aware deterministic runtime planner.
- Project-owned runtime mode file under `.project-governor/runtime/`.
- Clean reinstall helper to apply latest runtime mode to current/all governed projects.

## Behavior

- Does not copy plugin-global assets into projects.
- Does not read all initialization docs at session start.
- Uses `gpt-5.4-mini` for fast read-only scouting and `gpt-5.5` for high-quality implementation/review when available.
- Keeps micro-patches fast and subagent-free.
- Requires subagents for risky, upgrade, migration, and research workflows.

## Migration

Run:

```bash
python3 skills/clean-reinstall-manager/scripts/apply_latest_runtime_mode.py --path . --plugin-root /path/to/codex-project-governor --apply
```

Outside a project, use:

```bash
python3 skills/clean-reinstall-manager/scripts/apply_latest_runtime_mode.py --path . --plugin-root /path/to/codex-project-governor --discover-root "$HOME"
```
