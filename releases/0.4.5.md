# 0.4.5 — Hygiene Self-Inspection Fix

## Fixed

- `project-hygiene-doctor` now treats the Project Governor plugin repository itself as clean when `--project` matches `--plugin-root`.
- Root `.codex/agents`, `.codex/prompts`, and `.codex/config.toml` in the plugin source repository are classified as plugin repository runtime assets.
- Downstream projects that accidentally contain copied `.codex-plugin` or `skills` files are still flagged for manual review when they are not the configured plugin root.

## Compatibility

- No new dependencies.
- No JSON output shape changes.
- No project migration required.
