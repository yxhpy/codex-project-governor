## Claude Code

Use the installed Project Governor Claude Code plugin for governance workflows when available.

- Use `/pg-route` before non-trivial implementation.
- Use `/pg-context` before opening large governance documents.
- Use `/pg-quality` before final responses for coding work.
- Use `/pg-memory` when prior failures, repeated mistakes, stale memory, or durable learning may affect the task.
- Use `/pg-design` before UI/frontend edits.
